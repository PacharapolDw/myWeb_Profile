# myWeb_Profile
โปรเจกต์นี้เป็นเว็บไซต์ส่วนบุคคลที่พัฒนาขึ้นสำหรับรายวิชา Programming Web and Development
มีวัตถุประสงค์เพื่อแนะนำกีฬาที่ผู้พัฒนาสนใจ พร้อมอธิบายวิธีการเล่นและกฎของกีฬา ได้แก่ กอล์ฟ และ ฟุตบอล

# เว็บไซต์:
http://welcompd.mypressonline.com/mainWeb.php

# วัตถุประสงค์ของโครงงาน
เพื่อฝึกการพัฒนาเว็บไซต์ด้วย HTML, CSS และ PHP
เพื่อสร้างเว็บไซต์แนะนำกีฬาและการออกกำลังกาย
เพื่อฝึกการออกแบบ UI และโครงสร้างเว็บไซต์

# ฟีเจอร์หลักของเว็บไซต์
# หน้าแรก (Homepage)
แสดงข้อความต้อนรับ
มี Navigation Menu 4 เมนูหลัก
แนะนำภาพรวมของเว็บไซต์

# โปรไฟล์ (Profile)
แสดงข้อมูลส่วนตัวของผู้พัฒนา
แสดงทักษะและข้อมูลพื้นฐาน

# กีฬา (Sport)
แสดงกีฬาที่แนะนำ 2 ประเภท:
 - กอล์ฟ
 - ฟุตบอล
มีภาพประกอบและคำอธิบายเบื้องต้น

# เกี่ยวกับเรา (About Us)
แสดงข้อมูลเพิ่มเติมเกี่ยวกับเว็บไซต์
รายละเอียดหรือจุดประสงค์ของการสร้างเว็บ

# ภาษาที่ใช้
Frontend: HTML, CSS
Backend: PHP

/project-root

│── golf.php           # หน้ากีฬากอล์ฟ

│── /assets            # CSS, รูปภาพ, ไฟล์อื่น ๆ
